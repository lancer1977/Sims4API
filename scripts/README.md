# Scripts

## Entry points

- `deploy_sims4_target.sh` is the target-aware deploy helper. It supports
  `stream-box` first and `alienware-252` as the current live lane.
- `deploy_stream_box.sh` is the first-step target for `192.168.0.178`.
- `deploy_252_alienware.sh` preserves the current 252 lane wrapper.
